System Installer Manual

Interface Style: Simplified GTK2 (Zenity)

Description

AcreetionOS Lite features a streamlined, automated installer designed to get you up and running as quickly as possible. It minimizes the number of decisions you have to make by optimizing your system automatically during installation.

Using the Simplified Installer

The simplified installer launches automatically when you first boot into the live environment. To use it, follow these steps:

  1. Select Disk: Choose the hard drive where you want to install AcreetionOS. Warning: All data on this disk will be erased.
  2. Configure User: Enter your name, choose a username, and set a password.
  3. Install: Confirm the settings and wait for the progress bar to complete.
  4. Restart: Once finished, restart your computer and remove the installation media.

Advanced Installation

If you require more control over partitions or system settings, you can launch the Advanced Installer from the application menu at any time.
