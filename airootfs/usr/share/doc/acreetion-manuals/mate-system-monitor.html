Task Manager (System Monitor) User Manual

Private Sector Alternative: Windows Task Manager

Description

A tool to view and manage system resources and processes. It shows CPU, memory, and network usage history.

Basic Usage

  • View running processes in the 'Processes' tab.
  • Right-click a process to end or kill it if it becomes unresponsive.
  • Monitor resource usage graphs in the 'Resources' tab.
