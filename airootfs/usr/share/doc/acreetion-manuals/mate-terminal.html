Terminal User Manual

Private Sector Alternative: Command Prompt, PowerShell

Description

A terminal emulator for the MATE desktop environment. It allows you to access the command line interface of the operating system.

Basic Usage

  • Type commands and press Enter to execute them.
  • Use Ctrl+Shift+C to copy and Ctrl+Shift+V to paste.
  • Open multiple tabs with Ctrl+Shift+T.
