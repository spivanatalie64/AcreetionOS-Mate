Archive Manager (Engrampa) User Manual

Private Sector Alternative: WinRAR, 7-Zip

Description

An archive manager for the MATE environment. It allows you to create and modify archives, view the content of an archive, view a file contained in an archive, and extract files from an archive.

Basic Usage

  • Double-click a compressed file (zip, tar.gz, rar) to open it.
  • Click 'Extract' to unzip files to a folder.
  • Click 'New' to create a new archive from selected files.
