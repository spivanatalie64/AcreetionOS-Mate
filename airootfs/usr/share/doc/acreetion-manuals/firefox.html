Web Browser (Firefox) User Manual

Private Sector Alternative: Google Chrome, Microsoft Edge

Description

A powerful, open-source web browser that respects your privacy. It allows you to browse the internet, manage bookmarks, and install extensions.

Basic Usage

  • Type a website address in the top bar to visit a site.
  • Use Ctrl+T to open a new tab.
  • Use Ctrl+D to bookmark a page.
  • Access settings via the three horizontal lines in the top right corner.
