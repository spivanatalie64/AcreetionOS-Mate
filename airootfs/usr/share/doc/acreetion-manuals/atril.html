Document Viewer (Atril) User Manual

Private Sector Alternative: Adobe Acrobat Reader

Description

A simple multi-page document viewer. It can display PostScript (PS), Encapsulated PostScript (EPS), DJVU, DVI, XPS and Portable Document Format (PDF) files.

Basic Usage

  • Open a document via File > Open.
  • Use the toolbar to zoom in/out and navigate pages.
  • Use the sidebar to view the document outline or thumbnails.
