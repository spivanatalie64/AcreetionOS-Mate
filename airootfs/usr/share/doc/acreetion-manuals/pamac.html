Software Center (Add/Remove Software) User Manual

Private Sector Alternative: Microsoft Store, App Store

Description

A graphical package manager for installing, updating, and removing software. It supports official repositories and the Arch User Repository (AUR).

Basic Usage

  • Search for software using the search icon.
  • Click 'Install' or 'Remove' next to an application.
  • Click 'Apply' to execute pending changes.
  • Manage updates in the 'Updates' tab.
