Internet Connection Assistant

Utility Name: Internet Setup

Description

The Internet Connection Assistant automatically checks for an active internet connection on startup and whenever an installation process begins. If no connection is found, it helps you set one up.

Usage

  • Wired Connection: If you are plugged into an Ethernet cable, select "Wired" to automatically detect and connect.
  • Wi-Fi Connection: Select "Wi-Fi" to scan for available wireless networks. Choose your network from the list and enter the password if required.

You can also use the manual Wi-Fi Assistant from the system menu to change networks at any time.
