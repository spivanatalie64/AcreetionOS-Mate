Image Viewer (Eye of MATE) User Manual

Private Sector Alternative: Windows Photos, Preview

Description

A fast and functional image viewer. It supports many image formats including BMP, GIF, JPEG, PNG, TIFF, and ICO.

Basic Usage

  • Open an image file to view it.
  • Use the toolbar to rotate or flip the image.
  • View image properties (metadata) from the Image menu.
  • Start a slideshow from the View menu.
