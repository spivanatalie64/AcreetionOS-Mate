File Manager (Caja) User Manual

Private Sector Alternative: Windows Explorer, macOS Finder

Description

The official file manager for the MATE desktop. It allows you to browse directories, preview files, and launch applications associated with them.

Basic Usage

  • Navigate folders by double-clicking them.
  • Right-click files for options like Copy, Paste, and Delete.
  • Use the sidebar to quickly access common locations like Documents and Downloads.
  • Press F3 to split the view into two panes for easier file copying.
