Text Editor (Pluma) User Manual

Private Sector Alternative: Notepad, TextEdit

Description

A small and lightweight text editor for the MATE desktop. It supports standard text editing features and syntax highlighting.

Basic Usage

  • Type text directly into the main window.
  • Save files with Ctrl+S.
  • Open multiple files in tabs.
  • Use Search and Replace from the Search menu.
