Media Player (VLC) User Manual

Private Sector Alternative: Windows Media Player, QuickTime

Description

A free and open-source cross-platform multimedia player that plays most multimedia files as well as DVDs, Audio CDs, VCDs, and various streaming protocols.

Basic Usage

  • Open a media file via Media > Open File.
  • Use the playback controls at the bottom to play, pause, and stop.
  • Adjust volume with the slider or scroll wheel.
  • Full-screen mode can be toggled by double-clicking the video.
